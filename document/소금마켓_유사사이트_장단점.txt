1. 당근마켓

* 장점
-동네 직거래하기 편리
-동네 지역 커뮤니티 활성화
-무료 나눔
-사용자간의 평가제도가 있어 거래 매물의 신뢰도가 비교적 높음.
-직관적인 UI
-거래 신고 시스템 존재

* 단점
-내가 설정한 동네 범위 밖에서 거래 불가능
-앱 위치 기반 거래 플랫폼으로 웹에서 거래 불가능


2. 번개장터
* 장점
-웹과 앱에서 둘다 거래 가능
-키워드 연관 상품 추천
-거래 신고 시스템 존재

* 단점
-사용자 입장에서 택배 거래가 불편
-크롬 브라우저 환경에서 크로스브라우징 기능 작동안함
-사용자 친화적이지 않은 웹 UI


3. 중고나라

* 장점
-거래 매물 최대 규모
-포털 사이트 내 카페 서비스로 접근성이 좋음
-웹 환경에서 중고 매물 시세를 조회 가능
-신고 내역 조회 기능 존재

*단점
-사용자간의 평가 제도가 없어 거래의 신뢰도가 떨어짐
-네이버 카페 기반의 서비스로 게시판 수가 많음

=> 
* 채택 및 보완점
-중고 매물의 판매 시세(평균값 그래프)
-거래 신고 시스템
-직관적이고 고객친화적인 UI
-사용자간 평가 제도 도입
